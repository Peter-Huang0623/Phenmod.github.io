Mac下推荐的软件

Nov 17th, 2015 8:05 pm

传说中，Mac电脑是程序员专用的“飞机中的战斗机”，baidu一下“mac软件推荐”，你会得到很多优秀的工具。下面是我常用的几款，具体安装及使用方式不细说。

  • 终端工具 iterm，官网下载安装。preferences -> Keys -> HotKey，设置一个系统热键。（我的是cmd+shift+m）。

  • 终极 shell oh-my-zsh 。安装命令

      curl -L http://install.ohmyz.sh | sh

    具体使用参考池建强的文章

  • SourceTree ：git的图形化操作工具。

  • brew ：Mac下安装常用软件的利器。

      brew install <package_name>     #安装一个包
  brew update                     #更新 Homebrew 在服务器端上的包目录
  brew outdated                   #查看你的包是否需要更新
  brew upgrade <package_name>     #更新包
  brew cleanup                    #Homebrew 将会把老版本的包缓存下来，以便当你想回滚至旧版本时使用。但这是比较少使用的情况，当你想清理旧版本的包缓存时，可以运行
  brew list --versions            #查看你安装过的包列表（包括版本号）
  brew search <package_name>      #搜索

    brew cask : 优雅、简单、快速的安装和管理 OS X 图形界面程序。caskroom.io。 brew cask装的软件会放到 /opt/homebrew-cask/Caskroom 下面，然后连接到 ~/Applications。 跟系统默认安装位置 /Applications不一样。

      brew tap caskroom/cask          #添加 Github 上的 caskroom/cask 库
  brew install brew-cask          #安装 brew-cask
  brew cask install google-chrome     #安装 Google 浏览器
  brew update && brew upgrade brew-cask && brew cleanup #更新
  brew search <package_name>      #搜索想要的软件。

    一些我的安装包

      brew cask install appcleaner
  brew cask install cheatsheet
  brew cask install dropbox
  brew cask install onepassword
  brew cask install totalfinder

    强烈推荐装一些Finder里空格键快速查看文件内容的插件

      brew cask install qlcolorcode qlstephen qlmarkdown quicklook-json qlprettypatch quicklook-csv betterzipql webp-quicklook suspicious-package

    如果你经常使用 Alfred 启动程序，那么你会想 Alfred 可以搜索 brew cask 安装的程序，实现这些仅需运行：

      brew cask alfred link

    此外，你还可以通过 brew cask 安装 launchrocket，来管理通过 brew 安装的 service

  • Alfred ：自定义各种快捷操作，神器。

  • Moom ：快速设置窗口的位置及大小，全键盘流的福音。
  • Parallel Desktop ：想安装个虚拟机就用它吧，无缝连接系统，牛叉无比。
  • RescueTime ：关注你每天时间的使用，提高工作效率。
  • Charles ：抓包工具，可给真机做代理，可模拟网络不好的情况。
  • Mou ：写markdown文档。比如这篇。
  • Sublime Text ：你永远都不会知道这个看起来像个简单文本编辑器的编辑器有多牛逼。一堆堆的插件。比如写markdown想生成、更新目录，可以装个MarkdownTOC。支持Dash集成。
  • iTools ：不越狱查看真机app里的个各种文件。
  • XtraFinder ：增强Finder功能。
  • cheatSheat：长按 command ，将能查看当前程序的快捷键，好用到爆。
  • Sequel Pro：mysql，远端、本地数据库连接，方便易用
  • AppCleaner：应用程序卸载工具
